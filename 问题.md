## 当前代码状态

### stealth.cpp - MakeWindowUncapturable 函数

```cpp
void MakeWindowUncapturable(HWND hWnd) {
    typedef BOOL(WINAPI* pSetWindowDisplayAffinity)(HWND, DWORD);
    HMODULE hUser32 = GetModuleHandleW(L"user32.dll");
    if (hUser32) {
        pSetWindowDisplayAffinity pSWDA = (pSetWindowDisplayAffinity)GetProcAddress(hUser32, "SetWindowDisplayAffinity");
        if (pSWDA) {
            pSWDA(hWnd, 0x00000011);  // WDA_EXCLUDEFROMCAPTURE
        }
    }
    
    SetWindowLongW(hWnd, GWL_EXSTYLE, GetWindowLongW(hWnd, GWL_EXSTYLE) | WS_EX_TOOLWINDOW);
    
    SetWindowPos(hWnd, HWND_TOPMOST, 0, 0, 0, 0, SWP_NOMOVE | SWP_NOSIZE);
}
```

## 可能的问题方向（待排查）

1. **SetWindowDisplayAffinity**：`WDA_EXCLUDEFROMCAPTURE (0x11)` 可能影响消息传递
2. **WS_EX_TOOLWINDOW**：与某些窗口样式组合可能有问题
3. **HWND_TOPMOST**：置顶可能影响焦点
4. **键盘钩子拦截**：`WH_KEYBOARD_LL` 钩子可能干扰了消息循环
5. **消息循环问题**：`GetMessage` / `DispatchMessage` 可能有问题
6. **控件父窗口**：控件的父窗口句柄可能不正确
7. **窗口类注册**：`WNDCLASSW` 的某些字段可能有问题
8. **SS_NOTIFY 样式**：STATIC 面板控件可能拦截了鼠标消息
